AlphaSystem:
 - Status do projeto: Em desenvolvimento.
 - Tecnologias Usadas:
   - Java;
   - MySQL;
   - JDBC;
   - Java Swing.
- Desenvolvedor: Gustavo Henrique Bertassi.
  


Objetivo do software:

  O sistema AlphaSystem é desenvolvido para facilitar o cadastro de produtos, clientes, fornecedores, auxailiar no gerenciamento de uma empresa, assim como registrar de vendas e permitir a consulta do estoque da loja. Outro dos seus focos é a acessibilidade e na eliminação do uso de planilhas para a consulta das operações da empresa por meio do uso de banco de dados. Oque está presente no software é para o melhor fluxo de trabalho dos funcionários e armazenamento de dados de maneira mais segura e organizada.

Descrição do Projeto: 

  O AlphaSystem será composto com funcionalidades especificas para cada tipo de usuário. Pelo objetivo principal ser o de guardar dados de maneira mais organizada, será utilizado um banco de dados com estrutura de cadastro e de permissões para os usuários,  que ajudará na organização e na segurança mesmo com maior controle do fluxo de informações.
 
Registros no banco de dados:
  - Produtos
  - Clientes
  - Fornecedores
  - Vendas
  - Usuários


Funcionalidades do Sistema:
  - Cadastro de produtos
  - Cadastro de clientes
  - Cadastro de fornecedores
  - Cadastro de usuários
  - Registro de vendas
  - Consulta de estoque
  - Consulta de vendas
  - consulta de clientes
